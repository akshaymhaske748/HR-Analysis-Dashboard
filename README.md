# HR Analytics Dashboard – Power BI Project

## Overview
This project presents an interactive **HR Analytics Dashboard** built in **Power BI** to help organizations analyze workforce attrition, employee demographics, and job satisfaction patterns. The dashboard consolidates multiple HR metrics into a single visualization layer, enabling data-driven decision-making for HR teams and management.

## Tools & Technologies Used
- **Power BI Desktop**
- **Power Query** for data cleaning and transformation  
- **DAX** for measures and calculated columns  
- **Data Visualization** techniques including bar charts, donut charts, treemaps, and matrices  

---

## Project Objectives
- Identify the major drivers of attrition  
- Provide HR leadership with actionable insights  
- Understand demographic trends affecting retention  
- Explore satisfaction levels across job roles  
- Enable proactive workforce planning and targeted interventions  

---

## Insights Derived
- The **26–35** age group shows the highest attrition.  
- **Life Sciences** has the largest attrition count among education categories.  
- Employees with **≤ 5k salary** experience the highest attrition.  
- Certain job roles (e.g., **Laboratory Technician**, **Sales Executive**) contribute significantly to total attrition.  
- Tenure early in employment shows sharper attrition peaks.

---

## How to Use This Dashboard
1. Open the `.pbix` file in Power BI Desktop.  
2. Refresh the dataset using your HR data source.  
3. Filter by job role, department (Sales, HR, R&D), gender, and education fields.  
4. Review the key KPIs and visual summaries for strategic insights.
